# ArtTomato 개발 계획 체크리스트

이 문서는 [예술작품 정보 공유 리뷰 서비스 개요](/Users/shin/workspace/ArtTomato/docs/예술작품%20정보%20공유%20리뷰%20서비스%20개요.md) 와 [arttomato_stack.md](/Users/shin/workspace/ArtTomato/docs/arttomato_stack.md)를 바탕으로 만든 실행용 체크리스트다.

기준 원칙은 다음과 같다.

- 웹 MVP를 먼저 출시한다.
- 전시는 "현재 진행 중이거나 곧 열릴 전시"를 우선한다.
- 전시 정보는 자동 수집하되, 공개 전에는 관리자 승인을 거친다.
- 리뷰 기능과 전시 탐색 경험을 먼저 검증하고, 앱과 알림은 후속 단계로 둔다.
