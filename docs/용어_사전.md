# ArtTomato 용어 사전

서비스 문서/코드에서 동일한 의미로 사용하는 표준 용어다.

## 핵심 도메인

- 전시 (`exhibition`)
: 리뷰 대상이 되는 전시 단위.
- 장소 (`venue`)
: 전시가 열리는 미술관/박물관/문화공간.
- 작가 (`artist`)
: 전시 참여 작가.
- 태그 (`tag`)
: 전시 주제/장르/작가/키워드 분류값.
- 리뷰 (`review`)
: 사용자의 별점 및 한줄 총평.

## 운영/수집

- 관리자 승인 큐 (`pending_review`)
: 공개 전 검수 대기 상태.
- 수집 작업 (`ingestion_job`)
: 사이트 단위 배치 실행 이력.
- 수집 원문 (`ingestion_raw_item`)
: 목록/상세에서 추출한 원문 + LLM 정제 결과.
- 중복 전시
: 전시명 + 장소 + 기간 기준으로 같은 전시로 판단되는 항목.

## 상태값

- 예정 (`upcoming`)
- 진행 중 (`ongoing`)
- 종료 (`ended`)
- 숨김 (`hidden`)
- 검수 대기 (`pending_review`)
- 반려 (`rejected`)
