## 9) 수동 입력/복사 필요 항목 (이것만 준비하면 됨)

### A. Firebase 설정 파일 복사
1. Android Firebase 파일
- 필요 파일: `google-services.json`
- 가져오는 곳: Firebase Console > Android 앱(`com.subscri.manager`)에서 다운로드
- 복사 위치: `mobile/android/app/google-services.json`
- 주의: 파일의 package name이 현재 앱 `com.subscri.manager`와 일치해야 함

2. iOS Firebase 파일
- 필요 파일: `GoogleService-Info.plist`
- 가져오는 곳: Firebase Console > iOS 앱(`com.subscri.manager`)에서 다운로드
- 복사 위치: `mobile/ios/SubscriptionManager/GoogleService-Info.plist`
- 주의: Xcode에서 `SubscriptionManager` 타깃 포함(Target Membership) 확인

### B. Android Maps 키 입력
3. Google Maps API Key
- 필요 값: `GOOGLE_MAPS_API_KEY=실제키`
- 입력 위치(권장): `~/.gradle/gradle.properties`
- 대안(CI): CI 환경변수 `GOOGLE_MAPS_API_KEY` 또는 `MAPS_API_KEY`
- 주의: `mobile/android/gradle.properties`에는 실키를 커밋하지 않음

### C. 실행용 .env 값 입력
4. 루트/웹 환경 변수
- 파일: `.env`, `web/.env.local`
- 입력 필요 키:
  - `NEXT_PUBLIC_SUPABASE_URL`
  - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
  - `NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME`
  - `NEXT_PUBLIC_CLOUDINARY_API_KEY`
  - `NEXT_PUBLIC_CLOUDINARY_UPLOAD_PRESET`
  - `CLOUDINARY_API_SECRET`

5. 모바일 환경 변수
- 파일: `mobile/.env`
- 입력 필요 키:
  - `SUPABASE_URL`
  - `SUPABASE_ANON_KEY`
- 선택 키:
  - `CLOUDINARY_CLOUD_NAME`
  - `CLOUDINARY_UPLOAD_PRESET`
