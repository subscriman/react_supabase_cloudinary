# ArtTomato 서비스 기본 문구

## 1. 서비스 한 줄 소개

지금 열리는 전시를 실제 관람객 리뷰와 함께 찾는 서비스, ArtTomato.

## 2. 메인 페이지 소개 문구

전시 기간, 장소, 한줄 총평, 관람 팁을 한 번에 확인하고 방문 결정을 빠르게 도와드립니다.

## 3. 빈 상태 문구

- 전시 목록 없음: 현재 조건에 맞는 전시가 없습니다.
- 리뷰 없음: 아직 등록된 리뷰가 없습니다.
- 관리자 큐 없음: 현재 필터 기준으로 검수할 전시가 없습니다.

## 4. 로그인 유도 문구

리뷰 작성과 마이페이지 기능은 로그인 후 이용할 수 있습니다.

## 5. 관리자 검수 안내 문구

수집된 전시는 관리자 검수 후 공개됩니다. 승인 전 데이터는 대기 상태로 유지됩니다.

## 6. 고객 문의 기본 문구(초안)

- 문의 메일: support@arttomato.example
- 처리 시간: 영업일 기준 2~3일 이내 회신
- 신고/권리침해 요청: 문의 메일 제목에 `[신고]` 또는 `[권리침해]` 표기

## 7. 하단 고정 문구(푸터 초안)

ArtTomato는 전시 관람 결정을 돕기 위한 정보/리뷰 서비스이며, 전시 상세 정보는 기관 사정에 따라 변경될 수 있습니다.
